# 🚀 ResumeSpark -A Resume Builder App

🔗 **Live Demo:** https://saireddy-sm.github.io/ResumeSpark_app/

## 📖 Overview

ResumeSpark is a modern web application that helps users create professional, ATS-friendly resumes with ease. It provides a seamless resume-building experience through real-time preview, customizable templates, and PDF export functionality.

Designed with simplicity and usability in mind, ResumeSpark enables students, fresh graduates, and professionals to build polished resumes in minutes.

---

## ✨ Features

* 📝 Interactive Resume Builder
* 🎨 Multiple Professional Resume Templates
* 👀 Real-Time Resume Preview
* 📄 PDF Resume Export
* 💾 Local Storage Support
* 📱 Fully Responsive Design
* ⚡ Fast Performance with Vite
* 🎯 ATS-Friendly Resume Layouts

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/SaiReddy-sm/ResumeSpark_app.git
```

### Navigate to Project

```bash
cd ResumeSpark_app
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
