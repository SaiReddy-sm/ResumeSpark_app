\# Resume Builder App



A beginner-friendly Resume Builder Web App built with React and Vite.



\## Features



\- Resume input form

\- Live resume preview

\- Clean ATS-friendly resume layout

\- Download resume as PDF

\- Save resume data in localStorage

\- Clear saved data



\## How to Run



Open terminal inside the project folder:



```bash

cd resume-builder-app

npm install

npm run dev
